Searching for Component Sandboxes

In order to view your component sandboxes in the playground you'll need to first let the playground know what component you're looking for. So you'll need to open the command bar and then begin typing the name of the desired component.

The search results are based off matches against component class names. For example, in order to search for a component with a class name of something like `NoticeComponent`, you'd begin by typing the word `Notice`.

As you begin typing you'll notice that it's a fuzzy search finding both exact matches as well as other items that contain the characters from the search term.

  • The search will also match on the label property if one exists but will not match on the scenario descriptions.
