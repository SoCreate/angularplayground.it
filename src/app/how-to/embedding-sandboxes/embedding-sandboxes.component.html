Embedding Sandboxes

The Angular Playground works great as part of an overall style guide, component library, or documentation tool. For this reason it contains an "embed mode" that allows individual sandbox scenarios to be embedded in other applications with an `iframe`. When using this embed mode the src will include the code needed for the sandbox scenario only and will include nothing related to the command bar or anything else from the playground.

To use the embed mode you'll just need to add &embed=1 to the end of the url to your scenario.

.../?scenario=.%2Fapp%2Fnotice.component.sandbox/Default&embed=1
