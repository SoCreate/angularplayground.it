Installing

Angular Playground is available as an npm package.

``` npm install --save-dev angular-playground ```
Angular Playground depends on puppeteer to provide error-checking functionality for sandboxes. If you'd like to skip the automatic Chromium install, provide the following command before installing angular-playground:
``` npm set PUPPETEER_SKIP_CHROMIUM_DOWNLOAD true npm install --save-dev angular-playground ``` With Playground installed, your next step is to pick a guide based on your Angular setup. ### Pick a Guide: * If you're using @angular/cli * If you're using webpack
