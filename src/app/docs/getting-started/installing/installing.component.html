Installation

Looking for our developer guide?
#### Minimum Requirements * Angular: `5.0.0` * Typescript: `2.4.0` * Angular CLI (if applicable): `1.5.0`
#### Install from npm ``` npm install --save-dev angular-playground ``` With Playground installed, your next step is to pick a guide based on your Angular setup. #### Pick a Guide: * If you're using @angular/cli * If you're using webpack
