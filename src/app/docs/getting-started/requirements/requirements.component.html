Requirements

For Playground 2.x and above
Angular

* Requires Angular version `5.0.0` or above.

Typescript

* Requires Typescript version `2.4` or above.

Angular CLI

* If you are using Angular CLI you'll need to use version `1.5` or above.

compilerOptions Configuration
  • If you ARE using Angular CLI you'll need to change the module property in tsconfig.app.json to esnext.
  • If you ARE NOT using Angular CLI you'll need to change the module property in tsconfig.json to esnext.
