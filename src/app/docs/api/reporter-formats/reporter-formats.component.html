Reporter Formats

Playground currently supports two report formats: 1. `log` (default): Console log, no report file published 2. `json`: JSON output, defaults output file to `./sandbox.report.json` Specify a type with the `reportType` configuration flag: ``` angular-playground --check-errors --report-type json ``` If the provided format outputs a file, specify the file with `reportPath`: ``` angular-playground --check-errors --report-path ./reports.json --report-type json ``` Below is an example output file provided by the `JSONReporter`.
