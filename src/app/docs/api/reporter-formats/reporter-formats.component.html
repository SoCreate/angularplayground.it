Reporter Formats

Playground currently supports two report formats: 1. `log` (default): Console log, no report file published 2. `json`: JSON output, defaults output file to `./sandbox.report.json` 3. `xml`: XML output, defaults output file to `./sandbox.report.xml` Specify a type with the `reportType` configuration flag: ``` npx angular-playground --check-errors --report-type json ``` If the provided format outputs a file, specify the file with `reportPath`: ``` npx angular-playground --check-errors --report-path ./report.xml --report-type xml ``` Below is an example output file in JSON format.
