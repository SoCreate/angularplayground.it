The Playground CLI

Playground is designed to find files in your existing app with the `.sandbox.ts` extension that use the Playground API to set up sandboxes of components. To do this auto-discovery the Playground needs to create a static file on the fly (**sandboxes.ts**) to be able to have everything play nice with Angular's module system. To do that, Playground provides a cli! The cli for the playground can be run with the command: ``` angular-playground [config-file] ``` The `angular-playground` command will look for a file named `angular-playground.json` next to your `package.json` file by default. The `config-file` argument is optional and provides a way for you to point to the config file if you want to put it in a different location or name it something else. So, you will need a config file!
