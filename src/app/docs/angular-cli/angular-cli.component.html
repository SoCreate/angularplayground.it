Setting up an Angular CLI Powered Project

Playground ❤️'s Angular CLI and has built in support for integrating with it. If your app is using the **Angular CLI** you can set up your **angular-playground.json** file to have Playground make use of its `ng serve` under the hood.

When the `angularCli` property is present the `ng serve` command will be used when you run the `angular-playground` cli command. Below is a description of the properties of that object: + `envionment` *(optional)* The environment to target for serving up Playground (will tell Angular CLI to use this instead of its default) + `port` *(optional)* The port to tell `ng serve` to use. Defaults to `4201`. You will most likely want to create a new environment file (see the [angular cli docs on GitHub](https://github.com/angular/angular-cli#build-targets-and-environment-files)) for the playground and make some changes to your existing `main.ts` file and your existing environment files.
