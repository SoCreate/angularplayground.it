Switching Sandboxes From the Keyboard

When cycling through component sandbox scenarios in the command bar you can move between them with the up and down arrow keys.

up   — or —    down
Once a scenario is selected, the next time you open the command bar, hitting the `up` arrow key will move you to the scenario right before the currently selected item.
Likewise, hitting the `down` arrow key will move you to the scenario right after the currently selected item.
