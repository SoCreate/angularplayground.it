Opening the Command Bar

Once you have the playground installed, have sandboxed some components, and added some scenarios you're ready to get going. The playground contains a "command bar" that drops down from the top of the browser which can be opened by hitting ctrl + p at the same time or by hitting the f2 key.

ctrl + p   — or —    f2

The command bar will drop down with an empty search box that's used to find component sandbox scenarios.
