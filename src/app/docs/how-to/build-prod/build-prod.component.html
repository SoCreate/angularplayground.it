Note: This feature is highly reliant on @angular/cli and therefore is subject to change.
You can build Playground into a production application with the `--build` flag. #### Prerequisites * `@angular/cli` * `@angular/service-worker` #### Configuration Additionally, in order for Playground to use the `ServiceWorkerModule`, you'll need to create a custom Playground module with the following contents:
When bootstrapping Playground, use your custom module instead to enable Service Workers.
Finally, make sure that you have a valid `ngsw-config.json` provided within your project's `src`.

Build with a Service Worker

Assuming Playground is configured correctly, the following commands will build Playground with a service worker, navigate to the build output, and start serving content.

``` angular-playground --build cd ./dist/ http-server ```
Any http server application will do. This guide uses http-server.
Navigate to `127.0.0.1:8080` and open the "Applications" tab in Chrome Developer Tools to see the service worker in action.
