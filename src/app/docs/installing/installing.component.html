Installing

Angular Playground is available as an npm package.

``` npm i -D angular-playground ```

There are three primary steps to getting up and running with Angular Playground.

  1. Install the package into your project.
  2. Create a json config file for the Angular Playground CLI to use.
  3. Set up bootstrap logic to make use of the `PlaygroundModule` as an entry module.
From there you can start writing sandbox files for your components and directives and make use of the Angular Playground CLI to discover those files and watch for changes. With all that in place, you just need to serve up the bootstrap logic that uses the `PlaygroundModule`.
